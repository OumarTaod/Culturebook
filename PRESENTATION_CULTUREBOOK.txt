1. INTRODUCTION ET CONCEPT
================================================================================

TITRE DU PROJET : CultureBook - Réseau Social Culturel Guinéen

VISION :
- Plateforme dédiée à la préservation et au partage du patrimoine culturel Guinéen
- Connecter les communautés à travers leurs traditions, contes, proverbes et histoires
- Créer un héritage numérique pour les générations futures

MISSION :
- Immortaliser les traditions orales guinéennes
- Faciliter les échanges culturels entre régions
- Promouvoir la diversité culturelle guinéenne

3. ARCHITECTURE TECHNIQUE
================================================================================

FRONTEND (React + TypeScript)
- Framework : React 18 avec TypeScript
- Routage : React Router DOM
- État global : Context API
- Styles : CSS modules avec design system
- Responsive : 7 breakpoints (XL à Mobile Small)
- Notifications temps réel : Socket.IO

BACKEND (Node.js + Express)
- Runtime : Node.js avec Express.js
- Base de données : MongoDB avec Mongoose
- Authentification : JWT avec refresh tokens
- Upload fichiers : Multer
- Temps réel : Socket.IO
- Sécurité : Middleware de validation et protection

SÉCURITÉ
- Hashage des mots de passe (bcrypt)
- Validation des données d'entrée
- Protection CORS
- Gestion des erreurs centralisée
- Tokens JWT sécurisés
